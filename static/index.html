About RustMetrics

RustMetrics is a free, focused server and player tracker for Facepunch Rust. Data comes from Valve's standard UDP query protocols plus the BattleMetrics public API as a fallback for servers that disable A2S responses:

  • A2S for population, map, ping and player list of each watched server.
  • BattleMetrics for the live server browser and as fallback when a server blocks A2S.

Authentication runs over Steam OpenID 2.0 — you never enter your Steam password here. We only see your public SteamID64, display name and avatar.
